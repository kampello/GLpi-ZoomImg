# GLPI ZoomImg

O **GLPI ZoomImg** é um plugin para o GLPI que permite visualizar imagens em tamanho real com zoom e rotação diretamente na interface de chamados e documentos.

## 🚀 Funcionalidades
- Zoom em imagens anexadas nos chamados.
- Interface amigável para visualização rápida.
- Compatível com a visualização de documentos do GLPI.
